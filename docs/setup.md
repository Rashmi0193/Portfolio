# Project Setup

This document explains how this portfolio project was set up and why each tool was chosen.

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript
- Node.js

## Why Vite + React
- Vite provides a fast development server and minimal configuration.
- React allows building the UI using reusable components.
- This setup is commonly used in modern frontend projects.

## Initial Setup Steps
1. Created a new React project using Vite.
2. Started the development server to enable live reloading during development.
3. Installed Tailwind CSS for utility-first styling instead of writing custom CSS files.
4. Configured Tailwind to scan React files and generate only the required CSS.
5. Verified Tailwind setup by rendering a simple test UI.

## Development Workflow
- Run `npm run dev` to start the local server.
- Edit React components in `src/` and see changes instantly in the browser.
